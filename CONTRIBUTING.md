# Contributing

Thanks for your interest in improving ESP Mini Drone.

## Workflow
1. Fork the repository.
2. Create a branch named `feature/<short-description>`.
3. Make focused changes.
4. Add/update screenshots when geometry is changed.
5. Open a pull request.

## Commit Style
Use clear commit messages:
- `feat: add motor mount reinforcement`
- `fix: adjust top cover clearance`
- `docs: update print settings`

## Pull Request Checklist
- [ ] CAD changes reviewed in Fusion 360.
- [ ] STL exports updated in `cad/stl/`.
- [ ] Documentation updated in `README.md` or `docs/`.
- [ ] Screenshots refreshed if visuals changed.
