# Build and Print Guide

## Software
- Autodesk Fusion 360 (for editing `.f3z`)
- Any slicer software (Cura, PrusaSlicer, Bambu Studio)

## Open and Export
1. Open `cad/fusion360/New_Drone.f3z` in Fusion 360.
2. Verify component dimensions and clearances.
3. Export meshes to STL when needed.

## Suggested Print Settings
- Layer height: 0.16 mm to 0.2 mm
- Wall count: 3
- Infill: 20% to 35% (Gyroid recommended)
- Material: PLA/PETG
- Nozzle: 0.4 mm

## Parts
- `cad/stl/New_Drone_BODY.stl`
- `cad/stl/New_Drone_TOP.stl`

## Post Processing Checklist
- Remove supports carefully.
- Dry-fit top and body components.
- Verify motor mounting and wiring clearance.
